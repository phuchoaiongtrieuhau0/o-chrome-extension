Rule Collector Guide

Trang này hướng dẫn cấu hình rule collect dữ liệu trên web page. Rule được lưu theo domain trong selectorCollector:configs:v1.

Cách dùng nhanh

  1. Mở web page cần collect.
  2. Trong panel, vào tab Finder, chọn loại element như input/button/link rồi tìm theo text/value/placeholder.
  3. Bấm Use in builder để đưa selector vào form.
  4. Trong tab Builder, chỉnh label, mode, attr, multiple rồi bấm Add selector.
  5. Bấm Save rule, sau đó Dry-run rule để kiểm tra kết quả.

Schema selector

      
    
        
  • label: tên field dễ đọc.
    • 
        
  • selector: CSS selector dùng document.querySelectorAll.
    • 
        
  • mode=text: lấy textContent.
    • 
        
  • mode=attr: lấy attribute trong allowlist: value, href, src, alt, title, aria-label, data-*.
    • 
        
  • multiple=true: lấy tất cả element match; false chỉ lấy element đầu tiên.
    • 
        
  • trim=true: chuẩn hóa khoảng trắng.
